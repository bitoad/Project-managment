# Product Requirements Document (PRD)

## Product Name

Project Management ERP

---

# Vision

Build a professional web-based ERP platform for Engineering, Procurement and Construction (EPC) projects.

The platform should replace spreadsheet-based workflows with a centralized, collaborative, and scalable system.

---

# Target Users

- Project Manager
- Engineering Manager
- Procurement Engineer
- Site Supervisor
- Cost Controller
- Management
- Client (future)
- Vendor (future)

---

# Core Objectives

- Centralize project data
- Improve collaboration
- Track project progress
- Control costs
- Manage documents
- Generate reports
- Reduce manual work
